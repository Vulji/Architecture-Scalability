Le programme est un jeu dans le terminal qui gere la position de lettres représentant: un oiseau personnage jouable avec la lettre B et des tuyaux obstacle avec la lettre P. Dans la main loop on attend un input du joueur. Si le joueur presse entrer, la velocité de l'oiseau augemente ce qui le fait monter sur l'axe Y. Toujours dans la main loop les tuyaux sont déplacé sur l'axe X basé sur le delta time. Le programme check également si l'oiseau rentre en collision avec soit le sol soit un tuyau a chaque frame. A chaque fois que l'oiseau passe un obstacle, le programme change un string qui affiche le score, si le score est plus grand que le best score il updrate le string pour le best score. En collision l'oiseau est considéré mort et le best score est sauvegardé dans un txt file qui est load quand le programme est run a nouveau et update le string du best score par rapport à ce fichier.
